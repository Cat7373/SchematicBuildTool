# SchematicsBuildTool
在你的服务器建造用 Schematics "偷"来的建筑物

#使用要求:  
1. 服务器为 Bukkit 服务器.

#快速入门：  
1. 停止服务器.  
2. 将插件放进服务器的 plugins 文件夹.  
3. 启动服务器.  
4. 打开 plugins 文件夹中的 SchematicsBuildTool 文件夹.  
5. 将 Schematics "偷"来的建筑物存档放进去.  
6. 在游戏中执行 /sbt help 查看帮助.  
7. 在游戏中执行 /sbt build 文件名(不含后缀)来建造建筑.  
8. 可以修改 config.yml 里的建筑速度.
